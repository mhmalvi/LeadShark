## 🚀 Usage

### Setup

1. **Set OpenAI API Key:**
   ```bash
   # Windows
   set OPENAI_API_KEY=sk-your-key-here

   # Linux/Mac
   export OPENAI_API_KEY="sk-your-key-here"
   ```

2. **Optional: Configure LinkedIn credentials** (for better personalization):
   ```bash
   set LINKEDIN_EMAIL=your-email@example.com
   set LINKEDIN_PASSWORD=your-password
   ```

### Run Enrichment

```bash
# Run full enrichment with email sequences
python hybrid_ai_enhanced_enricher.py
```

## ⚠️ Known Limitations

1. **Requires OPENAI_API_KEY:** Email sequences will be skipped if API key not set
2. **Requires Minimum Data:** Name and company required for email generation
3. **Best With Full Enrichment:** Better personalization with LinkedIn data and Phase 4 results
4. **Rate Limits:** GPT API has rate limits (usually 50 req/min for Tier 1)

## 🐛 Troubleshooting

### Issue: "Email Sequence Status" = "Not Generated"

**Check:**
1. Is `OPENAI_API_KEY` set?
   ```bash
   echo %OPENAI_API_KEY%  # Windows
   echo $OPENAI_API_KEY   # Linux/Mac
   ```
2. Does row have "name" and "company" columns?
3. Check `hybrid_enricher.log` for errors

**Fix:**
```bash
set OPENAI_API_KEY=sk-your-key-here
```

---

### Issue: Generic email content

**Check:**
1. Are Phases 1-4 completing successfully?
2. Is LinkedIn integration working?
3. Is cold outreach generating pain points/activity?

**Fix:**
- Configure LinkedIn credentials for better profile data
- Ensure Google Search API for company info

---

### Issue: Empty email columns

**Check:**
1. Review Phase 5 logs for specific errors
2. Verify API key is valid
3. Check if previous phases failed

**Fix:**
- Set valid API key
- Review Phase 1-4 logs for failures
